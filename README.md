# Perpustakaan Umum Digital

Website perpustakaan umum digital dengan tampilan profesional dan struktur modern.

## Fitur
- Navbar dropdown lengkap
- Desain clean & responsif
- Struktur siap dikembangkan ke backend
- Cocok untuk Laravel / Node / API

## Cara Menjalankan
1. Clone repository
2. Buka file `index.html` di browser
3. Kembangkan backend sesuai kebutuhan
